Running OWASP Juice Shop

Running OWASP Juice Shop

Note: For the purpose of this lab the Juice Shop is already hosted on the remote server.

OWASP Juice Shop can be deployed in multiple ways:

  • One-click cloud instance deployed on Heroku
  • Local installation:
    • From sources
    • From pre-packaged distribution
  • Docker image
  • Vagrant
  • Amazon EC2 Instance
  • Azure Web App for Containers

Docker image

  1. Check if docker is up and running

    $ systemctl status docker
$ docker ps

  2. Pull juice-shop image

    $ docker pull bkimminich/juice-shop

  3. Run docker container

    $ docker run -d -p 3000:3000 bkimminich/juice-shop

  4. Browse to http://localhost:3000
